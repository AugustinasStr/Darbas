HTML ir CSS

CSS

(begginer)

CSS yra kalba, kurią naudojame HTML dokumento stiliui kurti.

CSS aprašo, kaip turi būti rodomi HTML elementai.

CSS pražia

Kad pradėti naudotis css iš pirmo reikia prašyti elementą kuri norite styliuoti. Klasės rašosi su "." pries elementa o id su "#"


        #id{(Styliavimas rašomas čia)
        }
        .klase{(Styliavimas rašomas čia)
        }
        body{(Styliavimas rašomas čia)
        }

Stiliai visalaika rašomi''{cia}'' ir baigiant vienai eilutei turite uždėti ";" kad galėtumete toliau pritaikyti stilių.

"Body" CSS 


    body{
        background-color: #0E2954;
        font-family: Arial, sans-serif;
    }
    _________________________________________________
    
    body {
        background-image: url('Fotkes URL');
        background-size: cover;
        background-position: center;
        background-repeat: no-repeat;
        font-family: Arial, sans-serif;
    }

  • background-image: url('Nuotraukos URL'); - Nuotraukos URL paversta i fona .

  • background-size: cover; - Fono dydis "cover" reškia kad 1 nuotrauka užimtu visa fona.

  • background-position: center; - Fono posicija centras.

  • background-repeat: no-repeat; - Kad fono nuotrauka kartotunis/nesikartotu.

  • font-family: Arial, sans-serif; - Raidžių šriftas.

"p" ir "h1" CSS


    h1 {
        text-align: left;
        flex: 1;
    }
    p{
    text-align: center;
    font-weight: bolder;
    letter-spacing: 5px;
    }

  • text-align: left; - Išlygina texta i kaire

  • flex: 1 - Padeda sukurt sudėtingus dizainus ir išdėstymus, kurie puikiai prisitaiko prie skirtingų ekranų dydžių.

  • text-align: center - išlygina texta į centra

  • font-weight: bolder - Padaro teksta stanbesni

  • letter-spacing: 5px -Tarp raidžių padaro 5pixeliu tarpa

Border CSS


    #border{
        background-color: red;
        border-radius: 10%;
        border-style: solid;
        border-width: 5px;
        width: 550px;
        height: 150px;
        box-shadow: 5px 5px 15px rgba(0, 0, 0, 0.5);

  • #border; - id pavadinta "border"

  • background-color: red; - fono spalva raudona

  • border-radius: 10%; - apibrežia kampų spinduli

  • border-style: solid; - Parenka borderio stiliu

  • border-width: 5px; - borderio linijos plotis

  • width: 550px; - Plotis

  • height: 150px; - Aukštis

  • box-shadow: 5px 5px 15px rgba(0, 0, 0, 0.5); - Padaro šešeli aplink borderi

Naudinga CSS


       .border{
       padding: 20px;
       margin: auto;
       gap: 40px;
      }

  • padding: 20px; - Tarpas 20px nuo kraštinės

  • margin: auto; - Apskaiciuoja kur padėti borderi

  • gap: 40px; - Tarpas tarp eilučiu ir stulpeliu 40px
