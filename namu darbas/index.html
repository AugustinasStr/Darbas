HTML ir CSS


HTML

(begginer)

viska reikia pradėti nuo !


        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta http-equiv="X-UA-Compatible" content="IE=edge">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Document</title>
        <link rel="stylesheet" href="style.css">
        </head>
        <body>
            
        </body>
        </html>

        
  •  <!DOCTYPE html> apibrėžia, kad šis dokumentas yra HTML5 dokumentas
    • 
        
  •  <html> yra pagrindinis HTML puslapio elementas
    • 
        
  • <head> yra metaduomenų talpykla
    • 
        
  •  <title> nurodo HTML puslapio pavadinimą (kuris rodomas naršyklės pavadinimo juostoje arba puslapio skirtuke)
    • 
        
  •  <link rel="stylesheet" href="style.css"> Elementas kuriame href galima prijungti css langeli
    • 
        
  • <body> apibrėžia dokumento turinį ir yra viso matomo turinio, pvz., antraščių, pastraipų, vaizdų, hipersaitų, lentelių, sąrašų ir kt., talpykla.
    • 
        
  • <h1> atsiranda didelę antraštę
    • 
        
  • <p> atsiranda pastraipą

Visi elementai turi prasideti ir baigtis taip pat bet pabaigoje elemento uzdetu reikia "/" pvz


        <h1>viskas eina čia</h1>
        <p>viskas eina čia</p>

Naudingi HTML elementai

Šie elementai praverčia, pavyzdziui kai norite dirbti su CSS div elementas labai praverčia


        <div>
        <p class="klase"></p>
        <p id="id"></p>
        <img src="https://gratisography.com/wp-content/uploads/2024/10/gratisography-cool-cat-800x525.jpg" alt="kietas katinas">
        <a href="https://www.freeformatter.com/html-escape.html#before-output">Spausk cia</a>
    

        
    
            
  • <div>-apibrėžia HTML dokumento padalijimą arba skyrių. Naudojama kaip HTML elementų talpykla.
    • 
            
  • class="klase"- atributas kuriame galima pažymėti klase, surinkti daugeli elementu i 1 klase.
    • 
            
  • id="id"- atributas naudojamas norint nurodyti unikalų elemento ID.
    • 
            
  •  <img src="Foto URL" alt="nuotrauka"> - elementas į kurį reikia ivesti nuotraukos url ir atsiras nuotrauka.
    • 
            
  • <a href="puslapio URL">Paryškintas tekstas</a>- tai paryškintas tekstas kuris nuveda i kita puslapi
