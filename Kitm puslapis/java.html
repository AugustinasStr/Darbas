    Java programuotojo mokymo programa

    Šios mokymo programos tikslas – parengti kvalifikuotą Java programuotoją, gebantį projektuoti ir kurti informacines sistemas bei nesudėtingas duomenų bazes, naudojant naujausias technologijas. Programa orientuota į komandinį darbą, pasitelkiant modernius programinės įrangos kūrimo įrankius.

    Front-End – interaktyvių vartotojų sąsajų kūrimas:

    Programa skirta parengti jaunesnįjį programuotoją, gebantį taikyti šiuolaikinius programinės įrangos kūrimo procesus bei išmanantį šias technologijas:

    • HTML5, CSS3, SASS – modernus tinklalapių dizainas ir stilių apipavidalinimas.
    • JavaScript (ES6) – dinamiškų ir interaktyvių tinklalapių kūrimas.
    • React, JavaFX – pažangios vartotojo sąsajos.
    • Bootstrap, Material CSS – karkasai greitam ir patogiam dizaino kūrimui.

      • Back-End – žiniatinklio, darbalaukio ir mobiliųjų aplikacijų kūrimas:

    • Programavimo pagrindai – kodavimo įgūdžiai ir algoritmų supratimas.
    • MySQL – reliacinių duomenų bazių kūrimas ir valdymas.
    • Java Spring – internetinių aplikacijų kūrimas.
    • Android programavimas – mobiliųjų aplikacijų kūrimas.
    • Darbalaukio programavimas – programų kūrimas asmeniniams kompiuteriams.
    • Turinio valdymo sistemos (WordPress) – svetainių kūrimas ir administravimas.
    • Git – versijų valdymas ir komandinio darbo įrankiai.

    Kaip vyksta mokymai?

    Vykdant mokymo programą taip pat mokinama operacinių sistemų, kompiuterių architektūros, Agile SCRUM metodologijos. Labai didelis dėmesys skiriamas praktiniam mokymui(-si), kuris sudaro daugiau nei 70% mokymosi laikotarpio. Gabiausi mokiniai turi galimybę praktines žinias gilinti realiuose projektuose bei dalyvauti meistriškumo konkursuose.

    Praktika ir karjeros galimybės

    Praktika vykdoma informacinių ir ryšių technologijų sektoriaus verslo įmonėse bei kitų sektorių organizacijų informacinių technologijų padaliniuose. Studentams taip pat suteikiama galimybė atlikti profesinę praktiką užsienio įmonėse per „Erasmus+“ programos stažuotes. Praktikos metu įgytos kompetencijos demonstruojamos, įtvirtinamos ir tobulinamos, siekiant paruošti studentus realiai darbo rinkai. Įgiję jaunesniojo Java programuotojo kvalifikaciją, absolventai gali įsidarbinti įmonėse ir organizacijose, kuriančiose mobilias ir darbalaukio aplikacijas, internetines informacines sistemas, tinklalapius, portalus, turinio valdymo sistemas bei elektroninės komercijos sprendimus. Baigus šią programą, mokslą galima tęsti kolegijose ar universitetuose, pasirinkus technologijų mokslų studijų srities programas.
