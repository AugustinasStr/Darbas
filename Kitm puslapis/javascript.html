    JavaScript programuotojas

    JavaScript programuotojo modulinė profesinio mokymo programa skirta parengti kvalifikuotą specialistą, gebantį kurti modernias ir interaktyvias programines sistemas, veikiančias vartotojų kompiuteriuose, išmaniuosiuose telefonuose, planšetėse bei kituose prieigos įrenginiuose (angl. front-end). Programoje taip pat mokoma sukurti sistemas, galinčias jungtis prie serverių ir duomenų bazių, bei valdyti JavaScript programavimo aplinką ir kūrimo procesą.

    Mokymosi tikslai ir įgūdžiai

    Norintys įgyti jaunesniojo JavaScript programuotojo kvalifikaciją įgyja šias kompetencijas:

    • Kurti pažangias WEB aplikacijas naudojant naujausias technologijas;
    • Taikyti objektiškai orientuotą programavimą su JavaScript;
    • Projektuoti, kurti ir naudoti duomenų bazes įvairaus sudėtingumo projektuose;
    • Vykdyti JS Unit testavimą ir užtikrinti kodo kokybę;

      • Naudoti Front-End technologijas:

    • HTML5, SASS, JavaScript, React JS, Redux;

      • Naudoti Back-End technologijas:

    • Node.js, NoSQL, MongoDB, Express karkasą;
    • Paruošti pažangią darbo aplinką su Docker.

    Praktika ir įsidarbinimo galimybės

    Praktika vykdoma informacinių ir ryšių technologijų sektoriaus verslo įmonėse bei kitų sektorių organizacijų IT padaliniuose. Studentams taip pat suteikiama galimybė dalyvauti „Erasmus+“ programos profesinėse stažuotėse, kuriose atliekama praktika užsienio įmonėse. Praktikos metu tobulinamos įgytos kompetencijos ir įgūdžiai.

    Karjeros perspektyvos

    Baigus programą, įgijęs kvalifikaciją asmuo galės dirbti jaunesniuoju JavaScript programuotoju IT įmonėse, kuriančiose interaktyvias internetines aplikacijas, mobilias platformas ar kitas skaitmenines sistemas.
